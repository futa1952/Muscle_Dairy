Muscle Diary

## サイト概要
体を鍛えている人、これから体を鍛えたい人に向けた筋トレSNSサイト
(イメージはインスタグラムの筋トレバージョン)

### サイトテーマ
筋トレ日記

### テーマを選んだ理由
自分が筋トレを始めようと思ってもいつも三日坊主で終わってしまう経験がある。
ただプログラミング学習は継続して続けられている。その差は毎日の目標設定と自分がやった事をアウトプットする事だと気付きこれらを日記のように出来るこのテーマにした。

### ターゲットユーザ
筋トレに興味がある10代〜40代

### 主な利用シーン
筋トレをする前に今日やる事を決める(目標設定)
筋トレ後に今日やった事のアウトプット
